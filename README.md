# Clipboard Bridge 📋 — iPhone ⇄ Windows クリップボード共有

同じ Wi-Fi につないだ **iPhone と Windows（や Mac）** の間で、コピー＆ペーストを
双方向に共有するツールです。Apple の「ユニバーサルクリップボード」を、
iPhone × Windows でも使えるようにするイメージです。

- PC でコピー → iPhone のアプリ画面に**リアルタイム表示**、ワンタップでコピー
- iPhone でコピー → **背面タップ2回**で PC のクリップボードへ送信
- 日本語・絵文字 OK / クラウドを経由しない（LAN 内で完結）/ 追加ライブラリ不要

---

## ⚠️ 最初に正直なこと（iPhone の制約）

iPhone は OS のルール上、**バックグラウンドで常にクリップボードを監視するアプリ**を
作れません（Apple が禁止しています）。そのため Mac↔iPhone のような「何もせず勝手に同期」
は iPhone 側では実現できません。

このツールは、その制約の中で**できる限りシームレス**にしたものです：

- **PC → iPhone** … アプリ（Web 画面）を開いていれば**自動で届く**。閉じていても「PCから取得」ショートカットで取得。
- **iPhone → PC** … **背面タップ**にショートカットを割り当てれば、コピー後に背面を2回叩くだけで送信。

慣れると「コピー → 背面トントン」だけなので、ほぼ純正の感覚で使えます。

---

## 1. 必要なもの

- Windows PC（または Mac）と iPhone が**同じ Wi-Fi**につながっていること
- PC に **Python 3** が入っていること
  - Windows: https://www.python.org/downloads/ から入れる際、最初の画面で
    **「Add python.exe to PATH」に必ずチェック**してインストール
  - Mac: 最初から入っています（`python3`）

---

## 2. Windows 側のセットアップ

1. この `clipboard-bridge` フォルダを PC の好きな場所に置く
2. **`起動-Windows.bat` をダブルクリック**
3. 黒い画面に次のような URL が出ます：

   ```
   ====================================================
     📋  Clipboard Bridge が起動しました
   ====================================================
     iPhone の Safari で次の URL を開いてください:

         http://192.168.1.23:8765      ← この数字は環境ごとに違います

   ====================================================
   ```

4. 初回は Windows が「このアプリの通信を許可しますか？」と聞いてきます。
   **「プライベート ネットワーク」にチェックして「アクセスを許可する」**を押してください
   （これを許可しないと iPhone から繋がりません）。

> この黒い画面は**開いたまま**にしておきます。閉じると同期も止まります。
> 使い終わったら画面を閉じる（または Ctrl+C）でOK。

---

## 3. iPhone 側のセットアップ（Web アプリ）

1. iPhone の **Safari** で、PC 画面に出た URL（例 `http://192.168.1.23:8765`）を開く
2. 共有ボタン（□↑）→ **「ホーム画面に追加」**

   これで**アプリのアイコン**としてホーム画面に並び、起動すると全画面で使えます。

### 使い方

- **PC でコピーしたもの**は、このアプリを開いていれば上のカードに自動表示。
  **「📥 iPhone にコピー」**を押せば iPhone のクリップボードに入ります。
- **iPhone から PC へ送る**ときは、下の欄に貼り付け／入力して **「📤 PC へ送信」**。

---

## 4. ★さらにシームレスに：背面タップで送受信（おすすめ）

iPhone の「ショートカット」アプリで2つ作り、**背面タップ**に割り当てると、
アプリを開かずにコピー＆ペーストを飛ばせます。

### ショートカットA：「PCへ送信」

1. ショートカットアプリ →「＋」→「新規ショートカット」
2. アクションを追加：
   - **「クリップボードを取得」**
   - **「URLの内容を取得」**
     - URL に `http://192.168.1.23:8765/clip` を入力（**自分のPCのIPに変える**）
     - 「詳細を表示」を開き：
       - **方法：POST**
       - **本文：要求の本文 → テキスト** を選び、値に変数 **「クリップボード」** を入れる
       - ヘッダを追加：キー `Content-Type` / 値 `text/plain`
3. 名前を「PCへ送信」にして保存

### ショートカットB：「PCから取得」

1. 新規ショートカット →
   - **「URLの内容を取得」** … URL `http://192.168.1.23:8765/clip`（GET のまま）
   - **「辞書の値を取得」** … キー `text`（入力は上の「URLの内容」）
   - **「クリップボードにコピー」** … 入力は上の「辞書の値」
2. 名前を「PCから取得」にして保存

### 背面タップに割り当て

`設定 → アクセシビリティ → タッチ → 背面タップ`

- **ダブルタップ** → 「PCへ送信」
- **トリプルタップ** → 「PCから取得」

これで、**iPhone でコピー → 背面トントン**で PC へ、
**背面トントントン**で PC の内容を iPhone へ、という操作になります。

---

## 5. セキュリティについて

- 通信は**自宅／職場の LAN 内だけ**で、クラウドや外部サーバーは一切経由しません。
- ただし**同じ Wi-Fi 上の他人からはアクセスできてしまう**ため、
  カフェ等の**公共 Wi-Fi では使わない**でください（自宅・自分のテザリング推奨）。
- パスワード保護が必要なら対応できます（下の「カスタマイズ」参照）。

---

## 6. うまく繋がらないとき

| 症状 | 対処 |
|---|---|
| iPhone から開けない | PC と iPhone が**同じ Wi-Fi**か確認。手順2-4のファイアウォール許可をしたか確認。 |
| しばらくして繋がらなくなった | PC の IP アドレス（URL の数字）が変わった可能性。黒い画面に出ている最新の URL を見る。 |
| IP が毎回変わって面倒 | ルーターで PC に固定 IP を割り当てると URL が変わりません。 |
| 文字化けする | 通常起きませんが、起きたら教えてください（ロケール対策済みです）。 |
| `python` が見つからない | Python を入れ直し、インストール時に「Add to PATH」にチェック。 |

---

## 7. カスタマイズ（任意）

- **ポート番号**を変える：`clipboard_bridge.py` 冒頭の `PORT = 8765` を編集
- **QR コードを表示**したい：PC で `pip install qrcode` すると、起動時に
  端末へ QR が出て iPhone のカメラで読み取れます
- **パスワード保護**や**インターネット越し（外出先）対応**もできます。必要なら声かけてください。

---

## 8. 「本当にダウンロードして起動するだけ」にしたい場合

今は「Python を入れて bat を起動」ですが、**Python 不要の単体 .exe**（ダブルクリックで起動）
にもできます。その場合は GitHub に置いて、リリースから .exe をダウンロード →
起動するだけ、という形にできます。希望があれば作ります。

---

作者メモ：依存ライブラリ無し（Python 標準ライブラリのみ）。Windows は Win32 API を
直接呼んで UTF-16 で読み書きするため、日本語も安全です。
